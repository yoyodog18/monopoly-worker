# Monopoly Game

A multiplayer Monopoly-style game built with modern web technologies.

## Tech Stack

- **Backend**: Supabase
- **Frontend**: Deployed on Cloudflare
- **Real-time**: Supabase Realtime for live game updates

## Project Structure

- `backend/` - Cloudflare Workers configuration
- `frontend/` - Frontend application
